Ce paquetage nommé Modèle reprend les classes Agressive, Cartes_RumeursV2, Joueur_Physique_ou_VirtuelV2, PartieV2 et l'interface Stratégie. Ce paquetage traite du Modèle de notre MVC, c'est ici que nous pouvons exécuter le code en lignes de commande. Nous ne sommes pas obligés de passer par l'interface graphique pour que le jeu fonctionne.
