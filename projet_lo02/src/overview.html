Le projet auquel correspondent les différentes classes et différents packages présents dans cette doc consiste à créer un jeu de carte virtuel nommé "Witch Hunt", dans lequel peuvent s'affronter des joueurs réels et virtuels
Le package Modele correspond à l'ensemble de nos classes, le package Vue correspond à l'interface graphique sur laquelle les joueurs peuvent jouer (le jeu est aussi jouable en lignes de commande) et le package Controleur correspond à notre cotrôleur, faisant le lien entre la vue et le modèle.
